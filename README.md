# WLog - 智能日记应用

一款支持 AI 助手、本地 RAG 检索、心情追踪的 Flutter 日记应用。

## 功能特性

### 核心功能
- **日记记录** - 支持文字、图片、心情、天气、饮食、体重等多维度记录
- **图片管理** - 支持多图上传、拖拽排序、全屏查看
- **日历视图** - 侧边栏日历快速导航，支持图片预览
- **搜索功能** - 快速搜索日记内容

### AI 助手功能
- **智能对话** - 支持 OpenAI/DeepSeek 等兼容 API
- **本地 RAG 模式** - 基于 BM25 的本地检索，完全离线，保护隐私
  - 智能意图识别 - 自动判断是否需要查询日记
  - 支持查询日记内容、心情统计、饮食记录等
  - Function Calling - 精确查询日记数据
- **对话记忆** - 保留最近 15 轮对话上下文
- **流式回复** - 实时显示 AI 回复内容

### 数据管理
- **数据导出** - 支持导出所有日记数据为 ZIP 文件
- **数据导入** - 支持从 ZIP 文件恢复日记数据
- **本地存储** - 所有数据本地存储，支持外部存储/应用存储切换
- **回收站** - 删除的日记可恢复

### 个性化设置
- **主题切换** - 支持浅色/深色/跟随系统主题
- **字体选择** - 多种字体可选
- **主题色** - 多种配色方案
- **每日提醒** - 可设置每日写日记提醒

### 其他功能
- **收藏功能** - 收藏重要日记
- **心情统计** - 查看心情分布趋势
- **饮食记录** - 记录每日三餐和零食
- **体重追踪** - 记录体重变化
- **时间自动插入** - 换行自动插入当前时间

## 技术栈

- **Flutter** - 跨平台 UI 框架
- **LangChain Dart** - AI 功能增强
- **Table Calendar** - 日历组件
- **Photo View** - 图片查看
- **Local RAG** - 基于 BM25 的本地检索（无需远程 API）

## 平台支持

- ✅ Android
- ✅ iOS
- ✅ Windows（手机分辨率模式 390x844）

## 安装运行

```bash
# 克隆项目
git clone <repository-url>

# 进入目录
cd diary_app

# 安装依赖
flutter pub get

# 运行应用
flutter run
```

## Windows 构建

```bash
flutter build windows
```

构建后的可执行文件位于 `build/windows/x64/runner/Release/diary_app.exe`

## AI 配置

1. 打开应用设置
2. 进入"AI 设置"
3. 配置 API 地址和密钥
   - 支持 OpenAI API
   - 支持 DeepSeek API
   - 支持任何 OpenAI 兼容的 API
4. 开启"允许 AI 访问日记数据"以使用 RAG 功能

## 隐私说明

- 所有日记数据本地存储
- 本地 RAG 模式完全离线，不发送数据到云端
- 普通模式仅发送必要数据到 AI API
